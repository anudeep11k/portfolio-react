# Portfolio React

## Build and Deployment

This project is built with Vite and React. Follow the steps below for local development and production builds.

### Installation
```bash
npm install
```

### Development
```bash
npm run dev
```

### Build
```bash
npm run build
```

### Deployment Commands
**Vercel / Netlify / Cloudflare Pages**
- Build Command: `npm run build`
- Output Directory: `dist`
- Framework Preset: `Vite`
